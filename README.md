# Student Management System

A REST API built using Spring Boot, Spring Data JPA, Hibernate, and MySQL.

## Features
- Create Student
- Get Student By ID
- Update Student
- Delete Student
- Get Passed Students
- Get Average Marks
- Get Top Student

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven

## API Endpoints

GET /stu/{id}

POST /stu

PUT /stu/{id}

DELETE /stu/{id}

GET /pass

GET /avg

GET /top
